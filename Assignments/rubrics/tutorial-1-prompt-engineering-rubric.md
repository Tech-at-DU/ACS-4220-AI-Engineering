# Tutorial 1: The Prompt Lab Rubric

Source: [../tutorial-1-prompt-engineering.md](../tutorial-1-prompt-engineering.md)

## Paste-Ready Markdown

```md
## Tutorial Completion

- **Needs Improvement**: Fewer than 5 chapters completed
- **Basic**: All 9 chapters completed; minimal experimentation
- **Proficient**: All chapters completed with evidence of experimentation beyond examples
- **Advanced**: Deep exploration; discovered edge cases or limitations

## Journal Quality

- **Needs Improvement**: Missing or surface-level entries
- **Basic**: 3 entries present but lack specificity
- **Proficient**: Entries show clear understanding; real scenarios are concrete and relevant
- **Advanced**: Entries demonstrate insight; identifies non-obvious applications

## Combined Prompt

- **Needs Improvement**: Missing or uses fewer than 3 techniques
- **Basic**: Combines 3 techniques but execution is rough
- **Proficient**: Clean combination of 3+ techniques with clear explanation
- **Advanced**: Sophisticated prompt that demonstrates mastery; would work in production

## Commit Quality

- **Needs Improvement**: Missing commits or disorganized
- **Basic**: Single commit with everything
- **Proficient**: Logical commits showing progression through the tutorial
- **Advanced**: Commit messages tell the story of your learning

```
