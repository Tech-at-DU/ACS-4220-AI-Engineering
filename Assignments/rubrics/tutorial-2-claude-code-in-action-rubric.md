## Paste-Ready Markdown

```md
## Course Completion

- **Needs Improvement**: Incomplete; missing modules
- **Basic**: All modules completed
- **Proficient**: Completed with evidence of engagement (notes, screenshots)
- **Advanced**: Completed on both platforms; compared experiences

## Workflow Audit

- **Needs Improvement**: Missing or generic observations
- **Basic**: Audit present but surface-level
- **Proficient**: Specific, honest comparison with concrete examples from own work
- **Advanced**: Deep analysis; identifies patterns in own workflow habits

## New Technique

- **Needs Improvement**: No new technique applied
- **Basic**: Attempted but incomplete implementation
- **Proficient**: Successfully applied one new technique with clear documentation
- **Advanced**: Applied multiple new techniques; measurable workflow improvement

## Reflection Quality

- **Needs Improvement**: No reflection on changes
- **Basic**: Brief mentions of what changed
- **Proficient**: Thoughtful before/after comparison with specific examples
- **Advanced**: Demonstrates genuine skill growth; would help a peer improve

```
